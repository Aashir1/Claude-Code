# Study Notes Creator

## Objective

Generate structured, academically rigorous study notes for a particular lesson that facilitate effective revision and reinforce key learning concepts.

## Instructions

When activated, this skill will:

1. **Analyze the lesson content** provided by the user
2. **Generate structured notes** following the academic format specified below
3. **Include revision elements** to reinforce learning and retention

## Required Output Structure

All generated study notes must adhere to the following format:

### 1. Summary at the Top
- Provide a concise overview (3-5 sentences) of the lesson's main concepts
- Highlight the learning objectives and key takeaways
- Establish context for the material covered

### 2. Key Terms Highlighted
- **Bold** all important terminology throughout the notes
- Provide clear definitions for technical or specialized vocabulary
- Use *italics* for emphasis on significant concepts
- Include etymological or contextual information where relevant

### 3. Main Content Organization
- Structure content using clear hierarchical headings
- Present information in logical sequence
- Include examples, analogies, and practical applications
- Incorporate visual formatting (lists, tables, code blocks) where appropriate

### 4. Revision and Reinforcement Elements
- Include memory aids (mnemonics, acronyms) where applicable
- Provide connections to previously learned material
- Highlight common misconceptions or areas of difficulty
- Suggest active recall strategies

### 5. Questions at the End
- **Comprehension Questions**: Test basic understanding
- **Application Questions**: Require applying concepts to new scenarios
- **Analysis Questions**: Demand critical thinking and deeper analysis
- **Synthesis Questions**: Encourage integration of multiple concepts
- Provide 5-10 questions spanning various cognitive levels

### 6. Easy to Read Formatting
- Use appropriate heading hierarchy (##, ###, ####)
- Employ bullet points and numbered lists for clarity
- Maintain consistent spacing and visual structure
- Include horizontal rules (---) to separate major sections
- Use code blocks for technical content
- Apply blockquotes for important principles or definitions

## Best Practices

- Maintain academic tone and precision in language
- Ensure factual accuracy and scholarly rigor
- Structure information to support progressive understanding
- Design questions that genuinely assess comprehension
- Format for both digital and print readability
- Include cross-references to related concepts where appropriate

## Usage Example

Invoke this skill when you need to:
- Create study materials from lecture content
- Summarize textbook chapters
- Prepare review notes for examinations
- Develop educational resources for independent study
- Convert complex topics into digestible learning materials
